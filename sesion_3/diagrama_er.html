Diagrama ER — Ejercicio Biblioteca

autores 1—N libros  ·  libros N—N categorias (vía libros_categorias)

erDiagram
    AUTORES ||--o{ LIBROS : escribe
    LIBROS ||--o{ LIBROS_CATEGORIAS : tiene
    CATEGORIAS ||--o{ LIBROS_CATEGORIAS : clasifica

    AUTORES {
        int id_autor PK
        varchar nombre
        varchar apellidos
        varchar nacionalidad
        date fecha_nacimiento
    }

    LIBROS {
        int id_libro PK
        varchar titulo
        varchar isbn
        year anio_publicacion
        int id_autor FK
    }

    CATEGORIAS {
        int id_categoria PK
        varchar nombre
        varchar descripcion
    }

    LIBROS_CATEGORIAS {
        int id_libro PK,FK
        int id_categoria PK,FK
    }
Relación 1-N: autores → libros
Relación N-N: libros ↔ categorias
